TITLE: Anti-Gravity Script Design — Multilingual Alphabet System

OBJECTIVE:
Design a unified, visually coherent alphabet system across multiple languages, incorporating symbolic, geometric, and conceptual consistency. The system should feel as if it is generated in a zero-gravity or anti-gravity environment, where forms are fluid, balanced, and spatially dynamic.

LANGUAGES TO INCLUDE:

1. Hebrew (22 letters + final forms)
2. Greek (24 letters)
3. Tamil (Uyir, Mei, Uyirmei structure)
4. Latin (A–Z)
5. Optional extensions: Arabic, Sanskrit, or futuristic symbolic scripts

CORE DESIGN PRINCIPLES:

* Anti-gravity aesthetics: floating strokes, suspended geometry, non-linear alignment
* Each letter must preserve its linguistic identity while adapting to the unified style
* Maintain readability while introducing futuristic abstraction
* Use consistent stroke logic, curvature rules, and spatial balance across languages

STRUCTURAL REQUIREMENTS:
For each language:

* Provide serial order of letters
* Include phonetic value
* Include symbolic meaning (conceptual or archetypal)
* Optional: assign numeric value (inspired by systems like Hebrew Gematria)
* Include a visual descriptor of each glyph (for rendering engines)

STYLE GUIDELINES:

* No heavy baselines; letters may float or orbit around invisible axes
* Use geometric primitives (circles, arcs, vectors) blended with organic curves
* Introduce depth illusion: layering, transparency, or soft shadows
* Maintain cultural respect: do not distort core recognizable shapes beyond interpretation

OUTPUT FORMAT:
For each language:

* Table format with columns:
  [Serial No | Original Letter | Stylized Form Description | Phonetic | Symbolic Meaning | Numeric Value (if applicable)]

Additionally:

* Provide a short design philosophy paragraph per language
* Provide a global design system explanation (how all alphabets connect visually)

ADVANCED FEATURES:

* Generate 2D glyphs and optionally 3D representations
* Suggest animation behavior (e.g., floating, rotating, pulsing)
* Define a color system (e.g., cosmic, monochrome, or energy-based gradients)

CONSTRAINTS:

* Avoid loss of recognizability
* Avoid cultural misrepresentation
* Ensure scalability for UI/UX usage (icons, fonts, educational tools)

FINAL DELIVERABLE:

* Structured multilingual alphabet system
* Visual design descriptions ready for rendering
* Consistent anti-gravity visual language across all scripts
